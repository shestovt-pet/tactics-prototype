# Тактический прототип

Изометрический пошаговый бой в духе Battle Brothers — в одном HTML-файле, открывается в любом браузере.

Ты управляешь **одним героем** (протагонистом) в отряде: рядом сражаются союзники, против вас — враги. Ходы идут по инициативе; всё, что меняет состояние боя, происходит только в свой ход. Между ходами сцепившиеся бойцы держат живую стойку — но урон наносится лишь в ходе.

## Как играть

Открой [`tactics.html`](tactics.html) двойным щелчком и играй:

| Действие | Как |
|---|---|
| Идти | клик по подсвеченной клетке (до 5 шагов) |
| Атаковать | клик по соседнему врагу (завершает ход) |
| Пропустить | кнопка «Завершить ход» |
| Заново | кнопка «⟳ Заново» |

Победа — когда пали все враги. Поражение — если пал ты или все союзники.

## Что уже есть

- Изометрическое поле 8×8, порядок по инициативе
- Управляемый только протагонист; союзники и враги под ИИ
- Шанс попадания (навык против защиты) и разброс урона
- Плавное перемещение, летящие цифры урона, искры и вздрагивание от удара
- «Живая стойка» сцепившихся бойцов между ходами (чисто визуальная)
- Победа/поражение, журнал боя, таблица инициативы

## Планы

Дальняя стрельба, броня, мораль — впереди.

## Служебные файлы

`server.ps1`, `.claude/launch.json` — локальный сервер для отладки, для игры не нужны.
