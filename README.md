# Прототипы боёвок

Набор игровых прототипов — по одному самодостаточному HTML-файлу на идею. Открываются в любом браузере, ставить ничего не нужно.

**Живая версия (GitHub Pages):** https://shestovt-pet.github.io/tactics-prototype/
Локально — двойной клик по [`index.html`](index.html) (каталог) или по любому прототипу.

## Прототипы

### Секторный бой — [`combat-sectors.html`](combat-sectors.html) · текущий
Бой без клеток. У бойца есть направление взгляда и **сектор внимания** (конус) как сенсор: приоритетной становится цель ближе к центру конуса; что вошло в конус — на то и реагирует. Ресурс **внимательности** тратится на шаги, довороты и способности (умение «перевести дух» восстанавливает), и **ширина конуса пропорциональна его остатку**: полный запас — обзор ~109°, пустой — туннельные ~34°. Выдохся — сам открылся для фланга и удара в спину. Есть **стрелки** с обеих сторон: бьют далеко, но только по цели строго внутри конуса, и держат эффективную дистанцию. У каждого бойца случайный **трейт** — Сильный / Ловкий / Умный (виден в никнейме): свои действия даются дешевле, а умный вдобавок видит шире и носит больше внимания. Состав — **до 10 бойцов на сторону**. Действие активного бойца проигрывается плавно и цельно, но имеет **цену в тиках** — и ровно на это время все остальные юниты одновременно и плавно работают ногами (держат дистанцию, кружат, берегут спину). Темп — ползунок «сек/такт». 5 способностей: сильный / быстрый / АоЕ-удар / перевести дух / кувырок. Любое изменение стейта проигрывается **строго по одному**, с вылетающим текстом и замедлением времени. Удар в спину — ваншот; ИИ бережёт спину. Настраивается состав (1–3 на сторону) и сколько союзников под ИИ (вплоть до полного автобоя).

### Боевой стенд — [`combat-bench.html`](combat-bench.html)
Реакционный ближний бой с физикой/инерцией: 3 сценария (1×3 / 3×1 / 3×3), доворот-с-внимательностью, оружие и видимые взмахи, направленные фигурки, импакт (hit-stop, тряска, вспышки).

### Тактика — [`tactics.html`](tactics.html) · архив
Пошаговый бой на изометрической сетке 32×32: отряд, по 3 умения на бойца (у героя 6), препятствия и линия огня, вражеский Вожак с приказами, диалоги. Отложенное направление, оставлено как отдельная ветка идей.
